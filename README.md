# Eleventy CV + Portfolio Starter


A minimal 11ty (Eleventy) site for a CV today and projects tomorrow.


## Quick start
```bash
# 1) Create repo and clone
# (ideally named <username>.github.io for GitHub Pages)


# 2) Install deps
npm install


# 3) Run locally
npm run dev
# Visit http://localhost:8080 (Eleventy dev server prints the exact URL)


# 4) Build
npm run build
# Output is in ./_site
my cv and portfolio
